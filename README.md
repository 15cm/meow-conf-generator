# MEOW Config Generator
A config generator for [MEOW](https://github.com/renzhn/MEOW) 

# Usage
1. Use `direct`, `proxy` directly. All under `./config-file-here`

2. Use `meow-conf-gen.py` to generate config files by yourself.
